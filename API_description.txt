GET и POST запросы. В ответ приходит JSON.
Обращение происходит следующим образом: example.com/api/method_name
Каждый запрос также возвращает логическое значение success, описывающее успешность операции
Если операция не успешна в ответ также будет включена строка err_msg

Методы API:

1. register - регистрация нового пользователя
Метод - POST
Параметры:
    login           string      Логин
    password        string      Пароль
Результат:
    Нет


2. auth - авторизация
Метод - POST
Параметры:
    login           string      Логин
    password        string      Пароль
Результат:
    session_key     string      Ключ сессии


3. channels_list - получение списка чатов, доступных для пользователя
Метод - POST
Параметры:
    session_key     string      Ключ сессии
Результат:
    channels        object[]    Массив объектов каналов
Объект канала: 
{
    id              int         ID канала
    name            string      Название канала
}


4. channels_add - добавить пользователю новый канал, используя его id
Метод - POST
Парамтры:
    session_key     string      Ключ сессии
    channel_id      int         ID добавляемого канала
Результат:
    Нет


5. channels_remove - удалить канал пользователю, используя его id
Метод - POST
Парамтры:
    session_key     string      Ключ сессии
    channel_id      int         ID добавляемого канала
Результат:
    Нет


6. channels_create - создать новый канал
Метод - POST
Параметры:
    session_key     string      Ключ сессии
    channel_name    string      Название нового канала
Результат:
    channel_id      int         ID нового канала


7. channels_delete - удалить канал
Метод - POST
Параметры:
    session_key     string      Ключ сессии
    channel_id      int         ID удаляемого канала
Результат:
    Нет


8. chat_history - получить историю сообщений чата
Метод - POST
Парамтры:
    session_key     string      Ключ сессии
    channel_id      int         ID канала
    count           int         Количество сообщений (не больше 50)
    offset          int         Смещение с конца (Например, если 70 - то 50 последних сообщений, не считая 70 последних)
Результат:
    count           int         Количество возвращаемых сообщений
    messages        object[]    Массив объектов сообщений
Объект сообщения: 
{
    author_id       int         ID автора сообщения
    author_name     string      Имя автора сообщения
    message         string      Текст сообщения
}


9. send_message - отправить сообщение в чат
Метод - POST
Парамтры:
    session_key     string      Ключ сессии
    channel_id      int         ID канала
    message         string      Текст сообщения
Результат:
    Нет
